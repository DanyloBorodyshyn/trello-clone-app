# First step

Use 
```bash
git clone https://github.com/DanyloBorodyshyn/trello-clone-app.git
```

# Second step

```bash
cd trello-clone-app
```

## Setup (Third step)

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Run app (Fourth step)

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
